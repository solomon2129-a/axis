# QUICK SETUP CHECKLIST

## Step 1: Update index.html
Add Firebase SDK to `<head>`:
```html
<script src="https://www.gstatic.com/firebasejs/10.7.0/firebase-app.js"></script>
<script src="https://www.gstatic.com/firebasejs/10.7.0/firebase-auth.js"></script>
<script src="https://www.gstatic.com/firebasejs/10.7.0/firebase-firestore.js"></script>
```

Update script tags at end of `<body>` (BEFORE closing </body>):
```html
<script src="auth.js"></script>
<script src="routing.js"></script>
<script src="auth-ui.js"></script>
<script src="script.js"></script>
</body>
```

## Step 2: Update style.css
Copy all CSS from `CSS_STYLES_TO_ADD.css` and paste at the END of `style.css`

## Step 3: Get Firebase Credentials
1. Go to https://console.firebase.google.com/
2. Create project or select existing
3. Enable Email/Password authentication
4. Create Firestore database
5. Copy config from Project Settings
6. Replace values in script.js (marked as YOUR_...)

## Step 4: Add code to script.js
Follow instructions in FIREBASE_SETUP.md - add initialization at top, auth listener at bottom

## Step 5: Test
1. `npm start` to run server
2. Open http://localhost:3000
3. Should see "Axis / Clutter → Clarity" start screen
4. Click Sign Up
5. Fill email/password, create account
6. Should redirect to journal page
7. Click "Axis" button (top-left) → see email + logout
8. Write something, click Make Sense
9. Go to Tasks page
10. Click logout → back to start screen

## Files Created:
- auth.js (authentication functions)
- auth-ui.js (UI interactions)
- routing.js (page navigation & access control)
- firebase-config.js (reference file)
- FIREBASE_SETUP.md (detailed setup)
- CSS_STYLES_TO_ADD.css (styles to add to style.css)

## Key Points:
✅ NO changes to existing journal/task pages
✅ Authentication completely separate
✅ Access control via auth state
✅ Floating dock only shows when logged in
✅ Axis menu shows user email
✅ All data saves to Firestore when user logged in
✅ localStorage still works as fallback

## Troubleshooting:
- If pages don't show, check browser console for errors
- If auth fails, verify Firebase credentials are correct
- If styles look wrong, make sure CSS was added to style.css
- Check that all .js files are in the same folder as index.html
